MAGAZINE

This is a magazine layout design project I have completed in a group at a graphic design course (IAT 102). In this course, we learned how to use Adobe software to create icon, magazine layout, and advertisement design. For this project, learning how to use basic design rules to set type and imagery was the main learning objective.

For the layout, the biggest challenge we faced was how to fit all of the 6-pages place-holder text and eight images into three spreads which ended up with a layout that was all packed with no white space and no hierarchy, as shown in my first draft. By thinking critically, we found out that giving more is not always good, so to solve the problem, we reduced the amount of images and decrease the font size and line spacing, which ended up with our final layout.

Through this project, I learned that too much information can not compose a good design. Comparing my draft and the final design, I realized how important simplicity and whitespace are in a design; they give reader a room to breathe and create better flow for audience to follow through, and even though a layout design cannot be interactive, it still should guide people through and give the content to the reader in an efficient way.
